## Description

Cloud Storage Learning Project with [NestJS](https://github.com/nestjs/nest) framework and TypeScript. This is a server part. Client code you can find [Here](https://github.com/Landerfire/cloud-storage_client).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
